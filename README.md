# 🧩 Study Project 1 — Slack Prototype (To-Do + Slash Commands)

A lightweight Slack-inspired chat interface with task management and command-based interactions.

## 💡 Key Features
- Chat-based interface
- To-do list management inside conversations
- Slash commands (e.g. `/add-task`, `/list`)
- Rapid prototyping using no-code / low-code tools

## 🛠 Tools

- Replit

# 🧚Study Project 2 — AI Fairy Tale Generator

A GenAI-powered application that creates personalized fairy tales based on user preferences.

## 💡 Key Features
- Guided onboarding flow
- Personalized story generation using AI
- Prompt optimization based on user input
- User authentication & account management
- Data storage for:
  - Preferences
  - Generated stories
  - Payment events
- Fake paywall (for experimentation & analytics)

## 🛠 Tech Stack
- Lovable (Frontend)
- Supabase
  - Database
  - Edge Functions
  - Authentication
- OpenAI API (text generation)

## 🧠 Product Thinking
- Focus on personalization via onboarding
- Experimentation with monetization (fake paywall)
- Structured data collection for iteration and analytics
